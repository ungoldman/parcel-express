# parcel-express

Starter kit for working with [`parcel`](https://parceljs.org) and [`express`](http://expressjs.com).

## Install

```
npm install
```

## Start

```
npm start
```

## Test

```
npm test
```

## Project Architecture

- `server.js` - server code (starts app)
- `index.html` - entrypoint for parcel
- `client.js` - browser code
- `style.css` - styles
